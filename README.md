# NANOCO Equipment Portal V4

Mục tiêu: bản hoàn thành nhanh trong tuần này.

Chức năng:
- 1 trang duy nhất: tìm kiếm → danh sách → lý lịch → lịch sử.
- Song ngữ Việt/Anh.
- Không hình ảnh, không manual, không PDF.
- Có thể nạp file Excel trực tiếp trên trình duyệt.
- Ưu tiên đọc 2 sheet: `Equipment_Register` và `Asset_Event_Log` hoặc `Asset_Event_Log1`.

Cách dùng:
1. Mở web.
2. Bấm `Nạp file Excel`.
3. Chọn file Excel quản lý thiết bị.
4. Web sẽ đọc danh mục thiết bị và lịch sử.
