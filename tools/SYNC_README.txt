NANOCO Sync Tool

Cách dùng nhanh:
1. Cài Python nếu máy chưa có.
2. Cài openpyxl: pip install openpyxl
3. Đặt file Excel vào cùng thư mục hoặc chỉ đường dẫn file.
4. Chạy:
   python excel_to_json_sync.py "Quản lý thiết bị.xlsx" "../data"
5. Upload/commit lại các file trong thư mục data lên GitHub.

Giai đoạn sau có thể đóng gói thành file .exe một nút SYNC.
